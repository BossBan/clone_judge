## Background Knowledge

In software development, copying existing code fragments and pasting them into other parts of the code is a common practice and a form of code reuse. The copied code is referred to as a code clone.

Code clones are classified as follows:

Type-1 (Exact Clone): The two code fragments are identical except for comments and whitespace.

Type-2 (Renamed Clone): The two code fragments are identical except for changes in variable, type, literal, and function names.

Type-3 (Modified Clone): The two code fragments are similar, based on Type-2 clones, with some statements added, deleted, or modified, as well as changes in code layout.

Type-4 (Functional Clone): The two code fragments implement the same functionality, but the implementation methods are different.

For example, bubble sort code and quick sort code implement the same functionality but in different ways, so they are Type-4 clones.
